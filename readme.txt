Happy Camper version 1.0 11/03/2015

GENERAL USAGE NOTES
--------------------
This is a project in progress for the fall 2015 cohort of coding camp Hackbright
Academy (session ends December 2015). 

The goal of Happy Camper is to be the Airbnb of camping gear. We want to
connect folks who want to rent gear with the people who have the gear (and
want to rent it out).


TECHNOLOGY STACK
--------------------
Application: Python, Flask, Jinja, SQLAlchemy, SQLite
APIs: Google Maps

FILES LIST
--------------------

model.py: table definitions
seed.py: fake data for tables
server.py: Flask app
